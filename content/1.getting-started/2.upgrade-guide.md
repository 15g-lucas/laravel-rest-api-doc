## Upgrade from v1 to v2

There is just a single breaking change here:

1. Add a search at the beginning of every front-end API call in the search body call:

```json
// BEFORE
// (POST) api/posts/search
{
  "filters": [
    {
      "field": "id", "operator": ">", "value": 1
    }
  ]
}

// AFTER
// (POST) api/posts/search
{
  "search": {
    "filters": [
      {
        "field": "id", "operator": ">", "value": 1
      }
    ]
  }
}

```
